# moutsTI

Repositório público destinado a **testes automatizados de Front-end e API**.  

Este projeto foi desenvolvido como parte de um **teste técnico da empresa moutsTI** e tem como objetivo demonstrar habilidades em:  

- Testes E2E para Front-end utilizando **Cypress**  
- Testes automatizados de API  
- Estruturação de projetos com **TypeScript**, classes e interfaces  
- Organização de pastas por escopo (frontend, backend, classes, interfaces)  

## Status do Projeto

✅ Projeto concluído e funcional.
